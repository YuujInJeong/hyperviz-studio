# DATAViz Studio

고급 데이터 시각화 및 회귀 분석 플랫폼

![HyperViz Studio Banner](https://via.placeholder.com/800x200/6366f1/ffffff?text=HyperViz+Studio+-+Advanced+Data+Visualization)

## 🎯 개요

HyperViz Studio는 다차원 데이터를 직관적으로 시각화하고 통계적 분석을 수행하는 웹 기반 플랫폼입니다. 최대 8차원까지의 데이터를 동적으로 탐색할 수 있으며, 고급 회귀 분석과 진단 기능을 제공합니다.

## ✨ 핵심 기능

### 1. 다차원 데이터 시각화

#### 기본 시각화 차원
- **X축, Y축**: 2D/3D 좌표계의 기본 축
- **Z축**: 3D 공간에서의 깊이 표현
- **색상**: 연속형 변수의 색상 매핑
- **크기**: 데이터 포인트의 크기로 변수 표현

#### 동적 차원 제어 (4차원 이상)
4차원 이상의 데이터는 슬라이더를 통해 동적으로 제어됩니다:

```
슬라이더 값 범위: [min, max]
스텝 크기: (max - min) / 100
허용 오차: max(range × 0.1, stdDev × 0.5, step × 3)
```

#### 자동재생 모드
- **순차적 모드**: 한 번에 하나의 차원씩 탐색
- **병렬 모드**: 모든 차원이 동시에 변화
- **개별 모드**: 각 차원이 독립적으로 변화

### 2. 고급 회귀 분석

#### 다중 선형회귀 모델
기본 회귀 방정식:
```
Y = β₀ + β₁X₁ + β₂X₂ + ... + βₙXₙ + ε
```

#### 모델 적합도 지표
- **결정계수 (R²)**:
  ```
  R² = 1 - (SS_res / SS_tot)
  ```
- **조정된 결정계수 (Adjusted R²)**:
  ```
  Adj R² = 1 - (1 - R²) × (n - 1) / (n - p - 1)
  ```
- **AIC (Akaike Information Criterion)**:
  ```
  AIC = 2k - 2ln(L)
  ```
- **BIC (Bayesian Information Criterion)**:
  ```
  BIC = ln(n) × k - 2ln(L)
  ```

### 3. 통계적 진단

#### 정규성 검정 (Shapiro-Wilk 근사)
```
W = (Σaᵢxᵢ)² / Σ(xᵢ - x̄)²
```
여기서 aᵢ는 정규분포의 기댓값 순서 통계량

#### 등분산성 검정 (Breusch-Pagan 근사)
```
BP = n × R²
```
여기서 R²는 잔차의 제곱을 설명변수에 회귀한 결정계수

#### 자기상관 검정 (Durbin-Watson)
```
DW = Σ(eᵢ - eᵢ₋₁)² / Σeᵢ²
```

#### 다중공선성 검정 (VIF)
```
VIFᵢ = 1 / (1 - R²ᵢ)
```
여기서 R²ᵢ는 Xᵢ를 다른 설명변수들에 회귀한 결정계수

### 4. 진단 그래프

#### 잔차 vs 적합값 플롯
```
Residuals = Y - Ŷ
```
선형성과 등분산성 가정 검증

#### Q-Q 플롯
```
Theoretical Quantiles vs Sample Quantiles
```
정규성 가정 검증

#### 척도-위치 플롯
```
√|Standardized Residuals| vs Fitted Values
```
등분산성 가정 검증

#### 영향점 분석 (Cook's Distance)
```
Dᵢ = (eᵢ² / (p × MSE)) × (hᵢᵢ / (1 - hᵢᵢ)²)
```
여기서 hᵢᵢ는 레버리지, p는 모수 개수

## 🔧 수학적 구현

### 감마 함수 근사 (Stirling's Formula)
JavaScript의 표준 Math 객체에 없는 lgamma 함수를 근사:
```
ln(Γ(z)) ≈ (z - 0.5) × ln(z) - z + 0.5 × ln(2π)
```

### t-분포 누적분포함수 근사
```
F(t) = 0.5 × (1 + sign(t) × √(1 - exp(-2t²/π)))  [df > 30]
```

### 데이터 필터링 알고리즘
슬라이더 값에 따른 동적 필터링:
```
|value - slider_value| ≤ tolerance
```
여기서 tolerance는 데이터의 분산과 범위를 고려하여 동적 계산

## 🎮 인터랙티브 기능

### 슬라이더 자동재생
- **프레임률**: 30 FPS로 부드러운 애니메이션
- **속도 조절**: 데이터 범위에 따른 적응적 속도
- **모드 전환**: 순차/병렬/개별 모드 지원

### 3D 시각화
- **회전**: 마우스 드래그로 자유로운 3D 회전
- **줌**: 마우스 휠로 확대/축소
- **팬**: 시점 이동

### 실시간 필터링
- **동적 허용오차**: 데이터 분포에 따른 적응적 필터링
- **성능 최적화**: 30fps 제한으로 부드러운 업데이트

## 🔒 프라이버시 및 보안

- **완전한 클라이언트 사이드 처리**: 모든 데이터가 브라우저에서만 처리
- **서버 전송 없음**: 데이터가 외부 서버로 전송되지 않음
- **로컬 저장**: 분석 결과는 브라우저에만 저장
